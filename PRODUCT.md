# Bittensor Subnet Intelligence System

## 1. Problem

Bittensor (TAO) has 75+ active subnets competing for ~2,880 TAO/day in emissions. Since Dec 2025, emissions are **100% flow-based** — net TAO staking inflows determine how much each subnet earns. 

**The opportunity**: Early identification of subnets with rising flow + solid fundamentals = alpha accumulation before the crowd. The challenge: no unified tool monitors flow trends, scores fundamentals, and generates actionable signals.

**Current state**: Investors manually check [Taostats](https://taostats.io), cross-reference subnet Discord/Twitter, and maintain spreadsheets. This is slow, incomplete, and misses inflection points.

## 2. Product Vision

An automated intelligence system that continuously monitors the Bittensor subnet ecosystem, detects flow trend changes in real-time, scores subnet fundamentals, and generates investment signals with evidence.

## 4. Core Concepts

### 4.1 Flow (Taostats)

**Flow = net TAO staking inflows (in - out)** per subnet, smoothed with a 30-day EMA.

- Determines emission share (100% as of Dec 2025)
- Negative flows set to zero
- Normalized across all subnets
- Only staking actions count — NOT registration, NOT emissions

**Signal**: Rising flow = growing conviction. Sustained flow = institutional interest.

### 4.2 Alpha Tokens

Each subnet has its own token. Price determined by:

```
Price = TAO_in / Alpha_in  (liquidity pool ratio)
```

Holding alpha earns:
- TAO emissions (proportional to emission share)
- Alpha emissions (distributed to stakeholders)

### 4.3 Emission Distribution (per block)

```
18% → Subnet owner
41% → Miners (via Yuma Consensus)
41% → Validators and their stakers
```

### 4.4 The Flywheel

```
More Flow → Higher Emission Share → More TAO in LP → Higher Alpha Price
    → Attracts more stakers → More Flow → (repeat)
```

## 6. Signal Framework

### 6.1 Flow Signals

| Signal | Detection | Meaning |
|--------|-----------|---------|
| **FLOW_SURGE** | Flow EMA 3-day slope > 2x 30-day avg slope | Sudden conviction spike |
| **FLOW_STEADY** | Flow EMA positive for 14+ consecutive days | Sustained accumulation |
| **FLOW_PEAK** | Flow EMA 7-day high but slope declining | Possible reversal |
| **FLOW_REVERSAL** | Flow EMA crosses below 7-day MA from above | Capital exiting |
| **FLOW_NEGATIVE** | Net flow < 0 for 7+ consecutive blocks | Subnet dying |

### 6.2 Fundamental Scores (Durbin Framework)

Each subnet scored 0-100 on:

| Dimension | Weight | Data Sources |
|-----------|--------|-------------|
| **Team Quality** | 20% | Doxxed? Track record? AI/Web3 experience? |
| **Use Case Clarity** | 15% | Clear problem statement? Service vs research? |
| **Technical Execution** | 20% | Uptime, incentive stability, GitHub activity |
| **Network Effects** | 15% | Miner count trend, validator count, unique users |
| **Community Sentiment** | 10% | Discord activity, Twitter engagement, SSI score |
| **Tokenomics Health** | 10% | LP depth, staking ratio, emission sustainability |
| **Competitive Moat** | 10% | Proprietary data/model? Defensible position? |

### 6.3 Composite Signal

```
SIGNAL = (
    flow_momentum_score * 0.35 +
    fundamental_score    * 0.35 +
    trend_alignment      * 0.15 +  # flow direction + fundamental direction aligned?
    risk_penalty         * 0.15    # regulatory, liquidity, inflation risks
)

OUTPUT:
  score >= 70  → BUY      (strong conviction)
  score >= 50  → ACCUMULATE (building position)
  score >= 30  → HOLD     (neutral, monitor)
  score >= 15  → REDUCE   (deteriorating)
  score <  15  → AVOID    (danger zone)
```

### 6.4 Evidence Chain

Every signal MUST include:
- Flow data (current EMA, 7d trend, 30d trend)
- Fundamental score breakdown
- Recent events (team updates, emission changes, new partnerships)
- Comparable subnets (peer analysis)
- Risk factors specific to this subnet

## 9. Risk Factors

### Systemic Risks
- **13% annual inflation** — TAO must appreciate >13%/yr for non-stakers
- **Regulatory** — SEC classification risk for TAO and subnet tokens
- **TAO correlation** — subnet tokens priced in TAO; TAO down = everything down
- **Emission decay** — 2nd halving Dec 2029 (0.25 TAO/block)

### Subnet-Specific Risks
- **Subnet failure** — deregistration possible (SN76 crashed 97%, SN103 in relegation)
- **Shallow liquidity** — extreme slippage on low-cap subnets
- **Team abandonment** — subnets are startups, teams can rug/ghost
- **Flow manipulation** — coordinated staking campaigns (expensive, 30-day EMA mitigates)

### Competitive Risks
- **Centralized AI** — OpenAI, Anthropic cheaper/faster/better for most users
- **Other DeAI** — Render ($696x P/rev), Akash ($32x), io.net (~2x)
